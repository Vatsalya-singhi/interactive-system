Interactive Systems

Goals :

  • The goals of UI designs are to place users in control of the interface as well as reduce cognitive load.
  • This application is built with the purpose of designing and developing a system where users can interact with similar UI components & elements, and upon user interactions multiple performance insights can be gathered and showcased.
  • The gathered data can be used to improve the overall design of the user interface and also better understand how users interact with it. This would be of great help to not just businesses, but also to individual users who wish to understand and improve their own user interface interactions as well as parameters such as user retention and establishing trust and recognition.
